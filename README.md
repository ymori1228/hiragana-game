# ひらがなをなぞろう

指やマウスでひらがなをなぞって練習できる、子ども向けのシンプルなWebゲームです。

## 遊び方

1. 画面に表示されたひらがなを、指（またはマウス）でなぞる
2. 「できた！」ボタンを押すと判定される
3. なぞれていれば花火が上がって次の文字へ進む
4. あ〜ん 全46文字をランダムな順番でクリアしたらゴール！

## 練習できる文字

**あいうえお・かきくけこ・さしすせそ・たちつてと・なにぬねの・はひふへほ・まみむめも・やゆよ・らりるれろ・わをん**（全46文字）

毎回シャッフルされるので、何度やっても異なる順番で練習できます。

## 使い方

`index.html` をブラウザで開くだけで動作します。インストール不要です。

```bash
open index.html
```

## 動作環境

- モダンブラウザ（Chrome / Safari / Firefox など）
- PC・スマートフォン・タブレット対応（タッチ操作対応）
